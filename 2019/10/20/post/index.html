How to set up a blog as github page with Hexo

Posted on 2019-10-20 Edited on 2019-10-27

Assumptions I’m making about you:

  • fairly familiar with git + gtihub

  • you want the blog to be available at
    http://username.github.io/projectname

  • Have none/ very basic hexo knowledge

If the above sounds right, click read more!

Install hexo

  • install hexo

  • setup a main repo for username.github.io
    don’t need a separate repo for the blog itself since hexo deploy will do that automatically

  • update _config.yml setup instruction

Very important! To setup blog as a project in your username.github.io page

  • add new post and deploy!
